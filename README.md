# neo-brutalism

Neo-brutalist portfolio recreation built with React + Vite + TypeScript.

## Highlights

- Strong neo-brutalist color system with `#FF4911` as primary accent
- Thick borders, hard shadows, and flat color blocks
- Responsive sections: hero, projects, about, contact, footer
- Interactive dynamics: hover/active/focus states, reveal on scroll, magnetic CTA motion
- Accessibility support with keyboard focus and reduced-motion fallback

## Scripts

```bash
npm install
npm run dev
npm run lint
npm run build
```
